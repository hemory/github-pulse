## Key Themes

### Theme 1: Manual Data Gathering (8/8 participants)
Every EM described a manual, time-consuming process. Tools mentioned: GitHub search (8/8), Jira (5/8), Slack scanning (6/8), spreadsheets (3/8). Average: 2-3 hours per week.

### Theme 2: Review Bottlenecks Are Invisible (6/8)
EMs learn about stuck PRs reactively. No proactive alerting exists.

### Theme 3: Trends Over Snapshots (8/8)
Universal preference for directional data over absolute numbers.

### Theme 4: Privacy Is a Feature (5/8)
EMs who have been burned by productivity metrics see privacy-first as a differentiator.

### Theme 5: Digest Over Dashboard (5/8)
Most EMs would consume a weekly summary more reliably than a real-time dashboard.

## Recommendations
1. Lead with weekly digest, dashboard as drill-down
2. Default to team-level aggregation
3. Emphasize trend arrows and anomaly detection
4. Build 1:1 prep as a first-class feature
